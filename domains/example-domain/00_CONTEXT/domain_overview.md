# Domain Overview

This file demonstrates the `lower_snake_case.md` naming convention required for files in `00_CONTEXT/`.

## Purpose

Context files provide background information, domain rules, and reference documentation that agents need to understand the domain.

## What Belongs Here

- Domain-specific rules and constraints
- Background information and context
- Reference documentation
- Glossaries and terminology

## Naming Convention

| Correct | Incorrect |
|---------|-----------|
| `domain_overview.md` | `DomainOverview.md` |
| `background_info.md` | `Background-Info.md` |
| `project_rules.md` | `ProjectRules.md` |
